Workshop Tools Setup

To prepare for your team workshop on Friday, install these tools on your own laptop: Visual Studio Code, Python, and GitHub Copilot Chat for VS Code. All labs will run locally on your current development device.

What To Install

  • Visual Studio Code
  • Python
  • GitHub Copilot Chat (VS Code extension)

Verification target: by the end, you can open VS Code, run python --version, and see Copilot Chat in VS Code.

1. Install Visual Studio Code

Windows

  1. Go to code.visualstudio.com.
  2. Click Download for Windows.
  3. Run the installer and keep default options.
  4. When prompted, keep these checked: add to PATH and add "Open with Code" actions.
  5. Launch VS Code from Start menu.

macOS

  1. Go to code.visualstudio.com.
  2. Click Download for macOS.
  3. Open the downloaded zip and drag Visual Studio Code.app to Applications.
  4. Open VS Code from Applications.
  5. In VS Code Command Palette, run Shell Command: Install 'code' command in PATH.

2. Install Python

Windows

  1. Go to python.org/downloads.
  2. Download the latest Python 3 installer for Windows.
  3. Run the installer.
  4. Important: check Add Python to PATH before clicking Install.
  5. Finish installation and open a new terminal.
  6. Run python --version and pip --version.

macOS

  1. Go to python.org/downloads/macos.
  2. Download the latest Python 3 macOS installer (.pkg).
  3. Open the installer and follow default steps.
  4. Open Terminal.
  5. Run python3 --version and pip3 --version.

3. Install GitHub Copilot Chat Extension

Windows

  1. Open VS Code.
  2. Open Extensions view (left sidebar, square icon).
  3. Search for GitHub Copilot Chat.
  4. Click Install on extension by GitHub.
  5. Sign in with your GitHub account when prompted.
  6. Open Copilot Chat panel and confirm it loads.

macOS

  1. Open VS Code.
  2. Open Extensions view (or press Cmd+Shift+X).
  3. Search for GitHub Copilot Chat.
  4. Click Install on extension by GitHub.
  5. Sign in with your GitHub account when prompted.
  6. Open Copilot Chat panel and confirm it loads.

Final Check Before Class

  1. VS Code opens successfully.
  2. Python command works: python --version (Windows) or python3 --version (macOS).
  3. Copilot Chat extension is installed and signed in.
  4. You can open this course site and access Notes, Discussions, and Uploads pages.
